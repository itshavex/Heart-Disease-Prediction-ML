## Folder Structure
- `papers/`: Empty directory reserved for manually storing downloaded IEEE research paper PDFs.
- `README.md`: This file, outlining the purpose, structure, and workflow.
- `literature_review.md`: Template for comprehensive literature review and comparative analysis.
- `methodology_notes.md`: Documentation of the project's technical methodology and pipeline decisions.
- `paper_summary.md`: Quick summary template for evaluating individual research papers.
- `research_progress.md`: Checklist tracker for project milestones.
- `future_work.md`: Ideas for extending the research, deep learning extensions, and clinical validation.
- `references.bib`: Centralized BibTeX repository for citations.

## Instructions for Adding New Papers
1. Download the research paper (PDF) from IEEE Xplore or another academic database.
2. Manually copy the PDF into the `papers/` directory.
3. Use a standardized file naming convention (e.g., `Year_Author_TitleKeyword.pdf`).
4. Fill out the `paper_summary.md` template for the paper to extract key insights.
5. Synthesize these insights into the main `literature_review.md` document.

## Citation Workflow
1. Locate the BibTeX citation for the downloaded paper (often provided by IEEE Xplore).
2. Append the BibTeX citation block to the `references.bib` file.
3. Ensure citation keys are consistent (e.g., `AuthorYearKeyword`).
4. Reference these keys in your final academic reports or methodology notes.
