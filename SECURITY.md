# Security Policy

## Supported Versions

We currently support the following versions of the Machine Learning pipeline with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Vulnerability Reporting

Security and responsible disclosure are critical to this research-grade ML project. If you discover a vulnerability or potential data leakage issue within the pipeline, please **do not** open a public issue.

Instead, report it directly to the repository author via private GitHub messaging or a connected professional network link (e.g., LinkedIn).

Please include:
- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact on the ML architecture or data integrity.

We will acknowledge receipt of your vulnerability report within 72 hours and strive to resolve the issue promptly in a secure release.
