# Contributing to Heart Disease Prediction ML

First off, thank you for considering contributing to this project. It's people like you that make the open-source Machine Learning community such a great place to learn, inspire, and create.

## 🚀 Project Setup
1. Fork and clone the repository.
2. Ensure you have Python 3.10+ installed.
3. Create a virtual environment: `python -m venv venv`
4. Activate it and install dependencies: `pip install -r requirements.txt`

## 📏 Coding Standards
- **PEP 8 Compliance**: All Python code must strictly follow PEP 8 style guidelines.
- **Docstrings**: Ensure every module, class, and function contains a descriptive docstring.
- **Type Hints**: Please use Python type annotations for all function arguments and returns.
- **No Unused Imports**: Keep imports clean and organized.

## 🔀 Branch Naming Convention
Please name your branches clearly based on the type of contribution:
- `feature/description` for new features
- `bugfix/description` for bug fixes
- `docs/description` for documentation improvements

## 🤝 Pull Request Guidelines
- Fill out the provided Pull Request Template completely.
- Ensure all existing ML metrics and reports still generate correctly (`python src/main.py`).
- Do not introduce breaking changes to the core `model_training.py` or `explainability.py` logic without opening an issue first for discussion.

## 🐛 Issue Reporting
If you find a bug or have a feature request, please open a detailed issue using our GitHub Issue Templates.
